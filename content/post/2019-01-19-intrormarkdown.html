Descripción:

Visión general

R Markdown proporciona un marco de autoría para la ciencia de datos. Puedes usar un solo archivo R Markdown para ambos

  • guardar y ejecutar código
  • generar informes de alta calidad que se pueden compartir con una audiencia

Los documentos R Markdown son completamente reproducibles y admiten docenas de formatos de salida estáticos y dinámicos (ver DIAPOSITIVAS) o (ver PDF).
